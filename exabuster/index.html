Exabuster is the Github Actions version of Buster, the bruteforce static site generator for Ghost.  Exabuster is still a work-in-progress, and not even close to done; use Buster instead!

Eventually Exabuster might allow one to bake out Ghost Blogs for Github Pages via Github Actions.
